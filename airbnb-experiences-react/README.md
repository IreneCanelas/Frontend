## Project Content

- JSX
- ReactDOM.render() and .createRoot()
- Custom components
- Organizing components
- Reusable components
- JS inside JSX
- Props
- Destructuring props
- Rendering arrays
- Mapping components
- Key prop
- Passing objects as props
- Spreading objects as props
